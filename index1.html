CSS Button Hover

UI Effect

An enhanced button hover effect:

/* Base button styles */ .btn { display: inline-flex; align-items: center; /* ... */ }
/* Primary button style */ .btn-primary { background: linear-gradient(135deg, #3b82f6, #ec4899); /* ... */ }
/* Hover effect */ .btn:hover { transform: translateY(-3px) scale(1.02); /* ... */ }

Enhanced CSS hover animation using transform and box-shadow.

Flexbox Centering

Layout

Centering an item horizontally and vertically:

Centered Item
.flex-container { display: flex; justify-content: center; align-items: center; /* ... */ }
.flex-item { padding: 15px; background-color: var(--secondary-color); /* ... */ }

Fundamental Flexbox centering technique using `justify-content` and `align-items`.
